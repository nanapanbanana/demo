# Магазин обуви

## Назначение программы

Приложение предназначено для просмотра товаров и оформления заказов в магазине обуви.

## Подготовка к запуску

Перед запуском программы необходимо:

1. Запустить Open Server.
2. Запустить сервер MySQL.
3. Импортировать базу данных из файла бд.txt.
4. Убедиться, что параметры подключения к базе данных указаны корректно (db.py).

# Запуск программы

1. Откройте папку с программой.
2. Запустите файл main.py
3. Выполните вход в систему или нажмите кнопку «Гость».

## Роли пользователей

### Гость

* Просмотр товаров.
* Поиск товаров.
* Сортировка и фильтрация товаров.

### Клиент

* Просмотр товаров.
* Оформление заказов.
* Просмотр своих заказов.

### Менеджер

* Работа с товарами и заказами.
* Поиск, фильтрация и сортировка.

### Администратор

* Добавление, редактирование и удаление товаров.
* Управление заказами.
(Чтобы изменить статус заказа, администратору необходимо нажать на заказ двойным нажатием кнопки мыши.)
